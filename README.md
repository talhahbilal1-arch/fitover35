# Fit Over 35

Fitness website for men over 35. Live at https://fitover35.com.

Static HTML/CSS site deployed via GitHub Pages.

## File Structure

```
fitover35/
├── index.html              # Homepage
├── 12-week-program.html    # 12-week training program
├── gear.html               # Gear recommendations
├── disclaimer.html         # Disclaimer
├── privacy.html            # Privacy policy
├── terms.html              # Terms of service
├── article-template.html   # Template for new articles
├── css/
│   └── styles.css          # All styles
├── articles/               # Published articles
│   ├── compound-lifts-guide.html
│   ├── strength-training-men-over-35.html
│   ├── protein-guide-over-35.html
│   ├── testosterone-boost-naturally.html
│   ├── lose-belly-fat-after-40.html
│   ├── best-home-gym-equipment.html
│   ├── build-muscle-after-35-limited-equipment-maximum-gains.html
│   └── alcohol-testosterone-what-men-over-35-need-to-know.html
├── CNAME                   # Custom domain (fitover35.com)
├── robots.txt              # SEO config
├── sitemap.xml             # Sitemap
└── 12-week-program.pdf     # Downloadable program PDF
```

## Deployment

Deployed via GitHub Pages from the `main` branch (root `/`).

Custom domain `fitover35.com` configured via CNAME file and DNS A records pointing to GitHub Pages IPs.

## Affiliate Links

Amazon affiliate tag: `fitover35-20`. Used in gear recommendations and article product links.

## Pinterest Integration

The site works with the Fit Over 35 Pinterest automation. Pinterest pins drive traffic to articles and the homepage.
